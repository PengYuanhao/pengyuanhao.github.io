flex布局

基本概念

  • flex布局为flexible box（弹性盒子）的缩写，用以为盒子提供最大的灵活性
  • 容器设置为flex布局后，项目的float、clear、vertical-algin将失效
  • 容器默认存在两根轴：水平主轴和垂直的交叉轴。主轴的开始位置为main start，结束位置为main end；交叉轴的开始位置为cross start，结束位置为cross end
  • 项目默认沿主轴排列，单个项目占据的主轴空间为main size，占据的交叉轴空间为cross size

项目属性

1. order

  • 描述

    ​ 定义项目的排列顺序。数值越小，排列越靠前，默认为0

  • 语法

2. flex-grow

  • 描述

    ​ 定义项目的放大比例，默认为0。也就是说即使存在剩余空间也不放大。若所有的项目的flex-grow都为1，则他们将等分剩余空间。若一个项目的flex-grow为2，其他项目都为1，则前者占据的剩余空间将比其他项多一倍。

  • 语法

3. flex-shrink

  • 描述

    ​ 定义了项目的缩小比例，默认为1，即如果空间不足，该项目将缩小。若所有项目的flex-shrink都为1，则当空间不足时，所有项目都将等比例缩小。若一个项目的flex-shrink为0，其他项目都为1，则当空间不足时，前者不缩小。

  • 语法

4. flex-basis

  • 描述

    ​ 定义在分配多余空间之前，项目占据的主轴空间。浏览器根据这个属性，计算主轴是否有多余空间。默认值为auto，相当于是项目本身的大小

  • 语法

5. flex

  • 描述

    ​ flex属性是flex-grow, flex-shrink 和 flex-basis的简写，默认值为0 1 auto。后两个属性可选

  • 语法

    建议优先使用这个属性，而不是单独写三个分离的属性

6. algin-self

  • 描述

    ​ 允许单个项目有与其他项目不一样的对齐方式，可覆盖align-items属性。默认值为auto，表示继承父元素的align-items属性，如果没有父元素，则等同于stretch。

  • 语法
